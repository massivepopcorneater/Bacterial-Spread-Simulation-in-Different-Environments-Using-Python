# Simulation of Bacterial Spread in Different Environments using Python
## Introduction
A program that simulates the spread of bacteria in different environments using Python. In this simulation, we account for temperature, pH and food variables. This program is made to be ran on home computers. Consequently, lower complexity methods will be used which sacrifices some realism. This project uses the Mesa Library an agent-based modelling library that allows for grid-based simulations, and Matplotlib, a library that allows users to create customized graphs. The application is divided into a file for the spread model, start program and agent/individual bacterium. This program aims to incorporate computer science, mathematics and biology concepts learned in CEGEP as well as research from peer-reviewed sources.
