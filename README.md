### Simulation of Bacterial Spread in Different Environments using Python
## Introduction
A program that simulates the spread of bacteria in different environments using Python. This project uses the Mesa Library for its grid-based simulation, agent-based modelling and Matplotlib because it allows us to create a graphical representation of the simulation. I divided the application into a file for the spread model, start program and agent/individual bacterium. This program aims to incorporate computer science, mathematics and biology concepts learned in CEGEP as well as research from peer-reviewed sources.
